Projektidee

Aenesche.de ist ein persönliches Webprojekt mit experimentellen Browser-Spielen. Die Seite kombiniert Design, UI-Experimente und Spielmechaniken mit JavaScript.

Gambling Simulation

Das Gambling-Spiel ist eine reine Simulation mit virtueller Währung. Es wird kein Echtgeld verwendet, keine Einzahlungen akzeptiert und keine Gewinne ausgezahlt. Das Spiel dient ausschließlich der Unterhaltung und dem Experimentieren mit Wahrscheinlichkeitsmechaniken.

AI Drone Lab

AI Drone Lab ist ein Incremental-Game im Stil eines Cookie-Clickers. Spieler bauen Drohnen-Produktionssysteme auf, erforschen Technologien und optimieren Skalierungsmechaniken. Ziel ist es, wirtschaftliche und exponentielle Systeme spielerisch darzustellen.

Technische Umsetzung

Beide Spiele laufen vollständig clientseitig im Browser. Spielstände werden lokal im LocalStorage gespeichert. Es existiert keine Benutzerregistrierung und keine serverseitige Konto-Verwaltung.

Werbung

Falls Werbung eingebunden wird, erfolgt dies über externe Werbeanbieter. Die Inhalte der Website sind unabhängig von Werbung und dienen primär als technisches und kreatives Projekt.

Project Idea

Aenesche.de is a personal experimental web project featuring browser-based games. The platform combines UI design experiments and simulation mechanics built in JavaScript.

Gambling Simulation

The gambling game is purely a simulation using virtual currency only. No real money is involved, no deposits are accepted, and no payouts are made. The game exists for entertainment and experimentation with probability mechanics.

AI Drone Lab

AI Drone Lab is an incremental game inspired by cookie-clicker mechanics. Players build drone production systems, research technologies, and explore scaling mechanics.

Technical Structure

All games run entirely client-side in the browser. Game progress is stored locally using browser storage. No user accounts or backend systems are implemented.

Advertising

If advertisements are displayed, they are served through third-party providers. The project remains independent and is primarily an experimental technical platform.
